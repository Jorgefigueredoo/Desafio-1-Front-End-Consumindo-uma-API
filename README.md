# Busca CEP

Projeto simples desenvolvido com **HTML, CSS e JavaScript** para consultar informações de um CEP usando a API pública do **ViaCEP**.

## Funcionalidades

- Buscar endereço pelo CEP
- Exibir rua, bairro, cidade e estado
- Validar se o CEP possui 8 dígitos
- Mostrar mensagem de erro caso o CEP seja inválido ou não encontrado

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- API ViaCEP

## Como funciona

O usuário digita um CEP no campo de busca e a aplicação faz uma requisição para a API do ViaCEP.  
Depois disso, os dados retornados são exibidos na tela.

## API utilizada

ViaCEP: https://viacep.com.br/
